# 🎯 CodeAlpha Tasks Repository

Welcome to the CodeAlpha Tasks repository! This is a collection of projects completed as part of the CodeAlpha internship program.

## 📁 Projects Included

### 1. ✅ Language Translator
A complete, beginner-friendly language translator application built with Python, Streamlit, and Google Translate API.

**Location:** `language-translator/`

**Features:**
- 🌍 100+ Language Support
- 🎨 Beautiful Streamlit web interface
- 🔑 No API Key Required (uses free Google Translate API)
- ⚡ Real-time Translation
- 📚 Python Basics Tutorial Included
- 📱 Mobile Responsive

**Quick Start:**
```bash
cd language-translator
pip install -r requirements.txt
streamlit run app.py
```

For detailed documentation, see: [`language-translator/README.md`](./language-translator/README.md)

## 🚀 Getting Started

### Clone the Repository
```bash
git clone https://github.com/thasmithagali-3679/codealpha_tasks.git
cd codealpha_tasks
```

### Run Language Translator
```bash
cd language-translator
pip install -r requirements.txt
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`

### Learn Python Basics (Optional)
```bash
cd language-translator
python python_basics.py
```
